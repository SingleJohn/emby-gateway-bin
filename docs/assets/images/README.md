# 截图素材命名建议

后续你补系统截图时，建议按这个规则命名，方便维护：

- `01-login-page.png`（登录页）
- `02-source-config.png`（Source 配置页）
- `03-backend-config.png`（后端配置页）
- `04-resource-pool.png`（资源池配置页）
- `05-route-rule.png`（路由规则页）
- `06-redirect-stats.png`（重定向统计页）
- `07-troubleshooting.png`（排障页面）

建议：

1. 统一宽度，避免页面跳动
2. 隐去敏感信息（域名、Token、密钥、真实路径）
3. 每次替换截图尽量保持同名，减少文档改动
