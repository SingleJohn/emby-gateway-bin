# 激活与许可证

Emby Gateway 使用许可证系统来管理功能访问权限。本文档将介绍如何激活许可证和续期流程。

## 激活许可证

### 1. 获取激活码

从 Emby Gateway 官方渠道获取激活码。

### 2. 执行激活

**操作步骤**：

1. 登录 Emby Gateway 管理界面
2. 点击左侧菜单中的「许可证」
3. 在「激活」卡片中，在「激活码」输入框中填写您的激活码
4. 点击「激活」按钮
5. 等待系统处理激活请求
6. 激活成功后，页面会显示许可证状态为「有效」

## 许可证续期

Emby Gateway 会自动处理许可证续期，您无需手动操作：

### 自动续期流程

1. **定期检查**：系统会定期检查许可证状态
2. **自动续期**：当许可证接近过期或处于宽限期时，系统会自动向许可证服务器请求续期
3. **状态更新**：续期成功后，系统会更新许可证状态和过期时间
4. **续期重试**：如果遇到网络问题，系统会自动重试续期操作

### 查看续期状态

**操作步骤**：

1. 登录 Emby Gateway 管理界面
2. 点击左侧菜单中的「许可证」
3. 在「许可证状态」卡片中，您可以查看：
   - **许可证状态**：当前许可证是否有效
   - **过期时间**：许可证的过期时间
   - **最后续期**：最后一次续期的时间

## 故障排查

如果遇到许可证相关问题，请参考 [排障指南](../ops/troubleshooting.md#83-许可证相关问题) 中的相关部分。

## 总结

激活许可证是使用 Emby Gateway 完整功能的必要步骤。激活后，系统会自动处理许可证续期，确保您的服务持续可用。如果遇到激活或续期问题，请联系 Emby Gateway 支持团队获取帮助。
