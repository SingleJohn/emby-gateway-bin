# 术语速查

给普通用户的术语表，不讲实现细节，只讲“它是干什么的”。

| 术语 | 简单解释 |
|---|---|
| Source | 播放请求进入网关的入口 |
| 后端 | 实际存放媒体文件的地方（S3/CDN/GDrive/本地） |
| 资源池 | 后端组合（主后端 + 备后端） |
| 路由规则 | 决定某类请求走哪个资源池 |
| 路径映射 | 把 Emby 路径转换成后端可识别路径 |
| 主备切换 | 主后端失败时自动尝试备后端 |
| Worker 模式 | 网关把 GDrive 流量转给 Worker 处理 |
| 后台 | 你日常配置和看状态的管理页面 |

## 推荐阅读顺序

1. `功能总览`
2. `快速开始`
3. `一步步配置指南`
4. `路由与故障切换`
